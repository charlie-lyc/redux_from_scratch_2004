# Redux from Scratch
Learn Redux from scratch with building 'Bugs Registry'


## Work Flow
- Design the store
- Define the actions <props> type, payload
- Create a reducer <arguments> state, action
- Set up the store


## Structure
- dist
    - index.html
- src
    - reducer.js
    - store.js : createStore()
    - index.js : dispatch(), getState(), subscribe(), unsubscribe()
    - actionTypes.js
    - actionCreators.js
- webpack.config.js
